\doxysubsection{Detailed Description}
A structure storing the placement parameters of a widget. 

\begin{DoxyVerb}    All go in pairs: a pointer to the value, and the value itself.
    If the pointer is NULL, this means that no parameter was provided and the default value
    should be used.
    If the pointer is not NULL, it points to the value field.
\end{DoxyVerb}
